# AC-20260703-postpoc-006 Plugin Distribution Trust Evidence

## Scope

Acceptance for `TASK-POSTPOC-006` /
`POSTPOC-PKT-006-plugin-distribution`.

Implemented a deterministic plugin distribution trust evaluator and install
manifest builder that models package manifests, signature verification,
revocation events, distribution indexes, allowlist references, compatibility
status, resource limits, migration notes, sandbox evidence, and installable vs
blocked plugin entries. It does not publish plugins or run a marketplace
service.

## Requirements

- `HATE-POSTPOC-GAP-006`
- `POSTPOC-PKT-006-plugin-distribution`
- `POST_POC_PRODUCTIZATION_DETAIL_SPEC.md#hate-postpoc-gap-006-plugin-distribution-and-trust`
- `WORKFLOW_COOKBOOK_OPERATION_CONTRACT.md`

## Acceptance Criteria

- Signed, allowlisted, compatible release plugins pass.
- Release and regulated profiles deny unsigned plugins.
- Release and regulated profiles deny unallowlisted plugins.
- Revoked plugins are denied.
- Package hash mismatch is denied.
- API migration-required plugins hold until migration evidence is present.
- Stale distribution indexes hold.
- Trusted compatible plugins can be emitted as installable manifest entries.
- Revoked, migration-required, stale, unsigned, unallowlisted, or unsafe
  sandbox plugins are blocked from install manifests.
- Plugin install manifests can be written as deterministic local artifacts.
