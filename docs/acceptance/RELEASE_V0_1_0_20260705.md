# HATE v0.1.0 PoC Preview Release - 2026-07-05

## Scope

This record supports the `v0.1.0` GitHub release as a PoC preview release of
the local-first HATE CLI. It does not claim production readiness, enterprise
readiness, regulated readiness, QEG approval, or final release authority.

Release posture:

- Package version: `0.1.0`
- Git tag: `v0.1.0`
- Release type: PoC preview
- `poc_ready`: true per `POC_COMPLETION_20260703.md`
- `product_ready`: false until external release/QEG approval and remaining
  production-readiness gaps are closed

major OSS two-cycle validation, platform operator workflow expansion, and docs
freshness verification.

Known caveats:
