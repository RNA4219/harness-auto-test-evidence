# AC-20260703-postpoc-010 Release Handoff Evidence

## Scope

Acceptance for `TASK-POSTPOC-010` /
`POSTPOC-PKT-010-release-handoff`.

Implemented a deterministic release handoff evaluator that records QEG,
Shipyard, Agent State Gate, and Agent Gatefield external verdict references.
It preserves denial outcomes and rejects any HATE attempt to claim final
release approval or overwrite an external verdict.

## Requirements

- `HATE-POSTPOC-GAP-010`
- `POSTPOC-PKT-010-release-handoff`
- `POST_POC_PRODUCTIZATION_DETAIL_SPEC.md#hate-postpoc-gap-010-qeg-and-shipyard-release-handoff`
- `WORKFLOW_COOKBOOK_OPERATION_CONTRACT.md`

## Acceptance Criteria

- HATE records external verdict refs and never converts them to self-owned
  release approval.
- Approved external references require both external run and decision refs.
- External denied or blocked statuses are preserved as hold.
- HATE final approval claims are denied.
- Verdict overwrite attempts are denied.
- Unavailable or unsupported handoff targets hold the report.
