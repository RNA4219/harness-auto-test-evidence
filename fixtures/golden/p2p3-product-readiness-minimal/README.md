# P2/P3 Product Readiness Minimal Fixture

P2/P3 は canonical HATE artifacts から product readiness と enterprise readiness の
advisory artifact を生成する。

この fixture は live SaaS、dashboard runtime、SSO、SIEM、warehouse connector を
起動しない。`product-readiness-report.json` は local artifact readiness の証跡であり、
QEG release gate や Shipyard publish approval を上書きしない。
